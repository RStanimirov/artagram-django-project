About Artagram Django Project

A note from the site's author (RS):

This website is the result of my Django project work during SoftUni's DjangoWeb module. It is created for the art community as a forum where users can interact with other people interested in Concept art, Graphic design and Art in general. I named this site Artagram because I thought the name should evoke the art theme it is devoted to. It is a Django project, i.e. it was developed with the Django web framework.

The site's users can do the following things:

  • Post artworks along with description and text. The logged-in users can update or delete their own content anytime.
  • The user can view other users' posts and artworks, and if logged-in, comment on them, like and/or dislike them.
  • Furthermore, the logged-in user can view a filtered list of all posts by a given user-author.
  • Special feature: Artagram has implemented a user rank system which endows the user with a status and icon based on their activity. More info can be found here.

Technical specifications:

  • Framework for creation of the website: Django
  • Database: PostgreSQL
  • Forms enhancer: Crispy forms
  • Images plugin: Pillow
