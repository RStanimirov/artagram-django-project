User Rank system

A special feature in this website is the implemented user rank system. When the users get registered, they start at the entry level - "Apprentice". The more posts they publish, the faster they climb on the rank ladder of Artagram. The available ranks are as follows:


I. "Apprentice" is the initial basic rank and is given this badge:

*The user gets the "Apprentice" rank upon registration. When the user creates more than 3 posts, the rank is upgraded to "Artisan".

II. "Artisan" is the next rank above "Apprentice" and is given this badge:

*The user gets the "Artisan" rank when the user creates more than 3 posts. When the posts reach 8, the rank is upgraded to "Craftsman".

III. "Craftsman" is the next rank above "Artisan" and is given this badge:

*The user gets the "Craftsman" rank when the user creates 8 posts. When the posts grow above 15, the rank is upgraded to "Master".

IV. "Master" is the highest rank in the Artagram rank system. It is above "Craftsman" and is given this badge:

* The user gets the "Master" rank when the user creates 15 posts. The "Master" rank is the highest rank in the Artagram rank system.
