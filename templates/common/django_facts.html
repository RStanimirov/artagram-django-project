What is Django ?

Django is an open-source Python-based web framework that helps to build web applications. This very website was developed with Django. The popularity of this framework increased rapidly in the last couple of years. Compare to other Python-based frameworks like Flask and Pyramid, the Django framework is far most popular among Python developers. It supports both Python 2.7+ versions and Python 3.x versions.
